# Changelog

## 1.0.0

Initial release.

- Color Rules color scheme: deterministic hash-based palette coloring, selectable in panel Standard options → Color scheme.
- Color Rules transformer: regex-based group coloring with optional line style overrides per capture group value.
- Secondary color (lighter shade) applied automatically to series matched by a dash line style override.
- Explicit per-rule color override: pin any rule to a specific palette color via the swatch picker in the transformer editor.
